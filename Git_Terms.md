# Git
- Git is a VCS( Version Control System).
- Created by **Linus Torvalds** in 2005
- Used to track history/ changes in code.

## Repository(Repo)
- It is a folder where git track your code or projects.

## Remote Repo
- A repository which is hosted on git hub.

## Clone
- Used to make a copy of a remote repo in to the computer.

## Staged
- It means code changes are save & file is now ready to commit.
- It is a holding or waiting area. 
- Use to add file.

## Commit
- Git take a snapshot of our changes with a massage. 
- file is now unchanged.

## Untracked
- New file that does not tracked by git.

## Modified
- File which is currently changed.

## Origin 
- This is the default name of main remote repo.

## Push
- Upload commits to the remote repo.

## Pull 
- Download and update changes from a remote repo in to local repo.

## Log
- Show commit history.
- Every commit generate a specific hash.

## Head
- Show current branch or commit.
- It is pointed by star (*main).

## Fork 
- Make a copy of someone's repo 

## Init
- Used to initialize git in a new folder.
