# Git Commands

## Set Username
- Here global specify that this username used for all repos.
```sh
git config --global user.name "name"
```

## Set User Email
```sh
git config --global user.email "email"
```

## Staged
- To add specific file
```sh
git add file_name
```
- To add all files
```sh
git add .
```

## Commit
```sh
git commit -m "massage_here'
```

## Push 
```sh
git push origin main 
```
## Init
```sh
git init
```
